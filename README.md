# React + TypeScript + Vite + FSD

This is a project template that is easy to start development with

## Getting Started

```
yarn install
yarn run dev
```

## Includes
- **React** 18
- **[FSD](https://feature-sliced.design/)** file structure. Eslint and aliases for it
- **TypeScript** 5
- Pre-configured **Eslint** and **Prettier**
